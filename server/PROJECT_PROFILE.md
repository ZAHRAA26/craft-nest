# ملف تعريفي للمشروع (Project Profile) - Crafts Platform Backend

## نظرة عامة
هذا المشروع هو الجزء الخلفي (Backend) لمنصة **Crafts Platform**، وهي منصة تهدف لربط الحرفيين (Artisans) بالمشترين (Buyers). المشروع مبني باستخدام تقنيات حديثة تضمن الأداء العالي والأمان وسهولة التوسع.

## التقنيات المستخدمة (Tech Stack)
- **Node.js & Express**: الإطار الأساسي لبناء خادم الويب والـ API.
- **TypeScript**: لضمان كتابة كود برمجى قوى وتقليل الأخطاء البرمجية.
- **MongoDB & Mongoose**: قاعدة بيانات غير علائقية لتخزين بيانات المستخدمين والمنتجات بمرونة عالية.
- **Passport.js (JWT Strategy)**: لتأمين المسارات والتعامل مع نظام تسجيل الدخول باستخدام الـ Tokens.
- **Zod**: للتحقق من صحة البيانات المدخلة من المستخدم (Data Validation).
- **Nodemailer**: لإرسال رسائل البريد الإلكتروني (تفعيل الحساب، استعادة كلمة المرور).
- **i18next**: لدعم تعدد اللغات (العربية والإنجليزية).
- **Swagger**: لتوثيق الـ API وتسهيل تجربة المطورين.
- **Helmet & CORS**: لتعزيز الأمان وحماية الخادم من الهجمات الشائعة.

## الهيكل التنظيمي للكود (Architecture)
يتبع المشروع نمط **Controller-Service-Model**:
- **Models**: تعريف جداول البيانات (User Schema).
- **Services**: تحتوي على منطق العمل (Business Logic) بشكل مستقل.
- **Controllers**: استقبال الطلبات (Requests) وإرجاع الردود (Responses).
- **Routes**: ربط المسارات بالـ Controllers.
- **Middleware**: وظائف وسيطة للتحقق من الصلاحيات، معالجة الأخطاء، والتحقق من البيانات.

## المميزات الحالية (Existing Features)
1. **نظام المستخدمين**:
   - تسجيل حساب جديد (مشتري أو حرفي).
   - تفعيل الحساب عبر البريد الإلكتروني.
   - تسجيل الدخول باستخدام (Access Token & Refresh Token).
   - استعادة كلمة المرور.
2. **نظام الإدارة (Admin)**:
   - عرض قائمة المستخدمين.
   - تعديل أدوار المستخدمين (Roles).
   - سحب صلاحيات الـ Refresh Tokens.
   - إرسال نشرة إخبارية (Newsletter) لجميع المستخدمين المفعلين.
3. **التدويل (Internationalization)**: دعم كامل للغتين العربية والإنجليزية في رسائل الخطأ والنجاح.
4. **التوثيق**: لوحة تحكم Swagger تفاعلية لتجربة المسارات.

## الحالة الحالية
المشروع في مرحلة الأساسيات القوية، حيث تم بناء نظام الأمان والهوية بشكل متكامل، وهو جاهز لإضافة المميزات المتعلقة بالمنتجات والطلبات.
