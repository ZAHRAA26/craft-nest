# 🚀 دليل تشغيل سكربت النشر السريع (Fast Deploy Guide)

هذا الدليل يشرح كيفية استخدام ملف `fast-deploy.sh` لتحديث الباك اند بضغطة واحدة.

## 1. الإعداد لأول مرة (مرة واحدة فقط)
يجب إعطاء جهازك صلاحية تشغيل الملف. افتح الـ Terminal في مجلد الباك اند واكتب:
```bash
chmod +x fast-deploy.sh
```

## 2. كيفية الاستخدام اليومي
كلما قمت بتعديل الكود وتريد رفعه للسيرفر، اتبع الخطوات التالية:

1.  افتح الـ Terminal في مجلد المشروع.
2.  قم بتشغيل السكربت:
    ```bash
    ./fast-deploy.sh
    ```
3.  سيسألك السكربت عن "وصف التحديث"، اكتب أي شيء (مثلاً: `fix login bug`) واضغط Enter.
4.  انتظر حتى تظهر لك الرسالة الخضراء ✅.

## 3. ماذا يحدث بعد التشغيل؟
بمجرد انتهاء السكربت، سيتم رفع الكود لـ GitHub. إذا كنت قد ربطت GitHub بمنصة **Render**، ستبدأ المنصة تلقائياً بسحب الكود الجديد وتحديث السيرفر. (يستغرق الأمر من 2-4 دقائق ليصبح التحديث حياً).
